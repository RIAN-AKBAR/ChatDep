## Cara Menggunakan:

1. **Buka Portal**: Akses `portal.html` di browser
2. **Klik Tombol Launch**: Tekan tombol "Launch WhatsApp Clone"
3. **Daftar**: Lalu gunakan akun demo yang Sudah di daftarkan
4. **Mulai Chat**: Pilih kontak dan mulai percakapan

## Fitur:
✅ Real-time messaging  
✅ File sharing  
✅ Responsive design  
✅ Multiple user accounts  
✅ Modern UI/UX
